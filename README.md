# pyre divers

Landing page for the Pyre Divers podcast — live, unpolished conversations with
the trailblazers making asymmetric bets. Two builders, one fire.

One self-contained `index.html`: a canvas whirlpool of thousands of tiny
hand-drawn stick figures riding a log-spiral current into a dark drain, over a
three-point ember gradient. The outer field animates; the ~8k bodies
concentrated at the center are baked once into a static layer. Honors
`prefers-reduced-motion`. No build, no dependencies.

Local: open `index.html` (or any static server). Deploys on Vercel as-is.
